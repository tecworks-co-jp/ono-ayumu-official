# 小野歩 オフィシャルサイト（案）

代表 小野歩氏の **著者・個人ブランディング** を目的としたオフィシャルサイトのプロトタイプ（提案版）です。
1枚完結（シングルページ）のHTMLで構成し、ヘッダーのメニューからページ内の各セクションへスクロール移動します。

最終更新: 2026-06-19 ／ ステータス: **提案・たたき台**

---

## 1. コンセプト

> **「IT人材の価値を、最大化する。」**

著者・経営者・コミュニティ主宰という3つの顔を1つに束ね、「IT業界で働く人のキャリアと可能性を広げる専門家・小野歩」としての **信頼性と権威性** を伝えることを最優先にしています。

- **トーン**: 編集的・上質・誠実（書き手＝著者としての知性を感じさせる）
- **キーメッセージ**: 「願望は知識から生まれる」「頼まれごとは試されごと」
- **想定読者**: 講演/取材/執筆の依頼者、メディア関係者、エンジニア・IT業界志望者、採用候補者、ビジネスパートナー

## 3. サイト構成（1ページ内のセクション）

すべて `index.html` の1ファイルに統合した **シングルページ構成** です。ヘッダーメニューは各セクションへのページ内アンカーになっています。

| アンカー | セクション | 役割 |
|----------|------------|------|
| `#home` | ホーム（Hero）＋実績数字 | 第一印象。キャッチ・肩書き・実績（著書3冊／相談1,000人／月3,000人／創業2022） |
| `#profile` | プロフィール | 経歴ストーリー・タイムライン・価値観・ミッション/ビジョン |
| `#books` | 著書 | 3冊の詳細 |
| `#media` | メディア・活動 | 活動領域・tecHub・メディア掲載一覧・SNS発信 |
| `#contact` | お問い合わせ | 依頼内容の明示＋フォーム（提案用ダミー） |

```
index.html（1ページ）
 ├─ #home     ホーム（Hero・実績数字）
 ├─ #profile  プロフィール（経歴・年表・価値観・理念）
 ├─ #books    著書（3冊）
 ├─ #media    メディア・活動（tecHub・掲載一覧・SNS）
 └─ #contact  お問い合わせ（フォーム）
```

## 6. プレビュー方法

ビルド不要の静的サイトです。リポジトリ直下の `index.html` をブラウザで開けばすべて確認できます。

```bash
# いずれかの方法で（リポジトリ直下で実行）
open index.html

# または簡易サーバーで
python3 -m http.server 8000
# → http://localhost:8000
```

## 7. 本番化の選択肢（参考）

| 方法 | 向き | 補足 |
|------|------|------|
| このHTMLをそのまま公開 | 早く・安く出したい | Netlify / Vercel / GitHub Pages に配置するだけ |
| 静的サイトジェネレータ化 | 更新を継続したい | Astro等。お知らせ追加を効率化 |
